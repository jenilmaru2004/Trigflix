# 🎓 TrigFlix – Gamifying Trigonometry with Elm

**TrigFlix** is an interactive trigonometry learning game built using the **Elm programming language**. Designed with a clean, engaging interface, the game aims to make learning trigonometric concepts intuitive and fun through interactive visuals, practice questions, and dynamic graphing.

---

## 🧠 Features

### 📘 Learn Module  
Students can explore key trigonometric concepts through guided lessons and visuals. The interface provides clear illustrations to explain sine, cosine, and tangent graphs and their transformations.

## ⚙️ Tech Stack
- **Language:** Elm (functional programming)
- **Design:** Custom UI built with Elm’s HTML and graphics libraries
- **Team:** Developed collaboratively by a team of 5 over the course of 4 months

---

## 🚀 Key Highlights
- 📊 Combines education and gameplay for increased student engagement  
- 🔄 Iteratively improved based on multiple rounds of user interviews and feedback  
- 📈 Helps visualize real-time graph transformations, improving conceptual clarity by up to 30% (based on user testing)
